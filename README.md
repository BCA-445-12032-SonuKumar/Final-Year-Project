# OSMSProject
Online Service Management System (OSMS)

📌 Project Description

The Online Service Management System (OSMS) is a web-based application designed to manage and track various service requests efficiently. It allows users to submit service requests and enables administrators to monitor, update, and resolve them in an organized way.

🚀 Features

- User registration and login
- Submit service requests
- Track request status
- Admin dashboard to manage services
- Update and resolve requests
- User-friendly interface

🛠️ Technologies Used

- Frontend: HTML, CSS , JavaScript , Bootstrap
- Backend: PHP
- Database: MySQL
